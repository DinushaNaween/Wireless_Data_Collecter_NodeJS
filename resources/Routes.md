# Routes

This file contains all the routes that are given through this api application.

## Role

1. Create role

    ```
    POST - '/role'
    ```

2. Get role by id

     ```
     GET - '/role/:roleId'
     ```

3. Get all roles

    ```
    GET - '/role'
    ```

4. Update role by id

    ```
    PUT - '/role/:roleId'
    ```

5. Delete role by id

    ```
    DELETE - '/role/:roleId'
    ```

6. Delete all role

    ```
    DELETE - '/role'
    ```

7. Disable role by id

    ```
    PUT - '/role/disable/:roleId'
    ```
